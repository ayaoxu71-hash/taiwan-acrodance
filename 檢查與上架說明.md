## 一、本機運作檢查

> ⚠️ 重要：請務必用「本機伺服器」開啟，不要直接雙擊 HTML 檔案。
> 直接用 `file://` 開啟會讓動態導覽列、公告欄失效（瀏覽器安全限制）。

### 啟動方式（三選一）

**A. VSCode Live Server 外掛（最推薦）**
1. VSCode 擴充功能搜尋「Live Server」並安裝
2. 對 `index.html` 按右鍵 →「Open with Live Server」
3. 瀏覽器自動開啟 `http://127.0.0.1:5500/index.html`
4. 存檔後瀏覽器會自動重新整理

**B. Python（免安裝套件）**
```
python -m http.server 8000
```
瀏覽器開 `http://localhost:8000`

**C. Node.js**
```
npx serve
```

## 三、上架（部署）

本網站為純靜態網站，可免費部署。以下提供兩種主流方式。

### 方式 A：GitHub Pages（免費、最常用）

1. 在 GitHub 建立一個新的 repository（例如取名 taiwan-acrodance）
2. 把整個專案資料夾的內容上傳：
   - 可用 GitHub 網頁版「Add file → Upload files」直接拖曳所有檔案
   - 或用 Git 指令：
     ```
     git init
     git add .
     git commit -m "上架網站"
     git branch -M main
     git remote add origin https://github.com/你的帳號/taiwan-acrodance.git
     git push -u origin main
     ```
3. 進入 repository → Settings → Pages
4. Source 選「Deploy from a branch」，Branch 選 `main`、資料夾選 `/ (root)`，按 Save
5. 等待約 1～2 分鐘，網站會發布到：
   `https://你的帳號.github.io/taiwan-acrodance/`

> 注意：GitHub Pages 網址帶有子路徑（/taiwan-acrodance/）。
> 本網站所有連結都用「相對路徑」（如 about.html、assets/css/style.css），
> 在子路徑下可正常運作，無需修改。

### 方式 B：Zeabur（可綁自訂網域，原本就在用）

1. 前往 zeabur.com 登入
2. 連接你的 GitHub repository
3. 選擇「靜態網站 / Static」類型
4. 部署後可在設定中綁定自訂網域（如 taiwanacrodance.zeabur.app 或自有網域）

### 方式 C：其他免費平台（擇一即可）
- Netlify：直接把整個資料夾拖到 netlify.com 的部署區即可
- Cloudflare Pages：連接 GitHub 後自動部署
- Vercel：連接 GitHub 後自動部署
